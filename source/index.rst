Installation
------------

To install Kavy, follow the instructions in the [installation guide](installation.rst).

Usage
-----

Learn how to use Kavy and its chatbot by reading the [usage guide](usage.rst).

Configuration
--------------

Configure Kavy by modifying the [config.ini](config.ini) file. Refer to the [configuration guide](configuration.rst) for details.

PDF Module
----------

The PDF module in Kavy allows you to process PDF data. Read more in the [PDF module documentation](modules/pdf_module.rst).

Word Module
-----------

The Word module in Kavy allows you to process Word document data. Read more in the [Word module documentation](modules/word_module.rst).

URL Module
----------

The URL module in Kavy allows you to process data from URLs. Read more in the [URL module documentation](modules/url_module.rst).

TXT Module
----------

The TXT module in Kavy allows you to process plain text data. Read more in the [TXT module documentation](modules/txt_module.rst).

Contributing

If you'd like to contribute to Kavy, please follow the guidelines in the [contributing guide](contributing.rst).
